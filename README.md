Instagram'daki Tüm Takipçilerinizi Takip Etmemek İçin En İyi Araç.

## Gerekliler
1. openssl
2. curl

## Termuxta Nasıl kurulur

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/egiusta/unfollow`

`$ cd unfollow`

`$ chmod +x insunfollow.sh`

`$ termux-wake-lock`

`$ bash insunfollow.sh`
