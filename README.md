## Requirements
1. openssl
2. curl

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/termuxprofessor/insunfollow`

`$ cd insunfollow`

`$ chmod +x insunfollow.sh`

`$ termux-wake-lock`

`$ bash insunfollow.sh`
